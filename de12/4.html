タイピングゲーム

ゲーム説明

・ランダムでことわざが出題
・制限時間100秒
・エンターでスタート
・リトライ可能

ソースコード

  1. import random
  2. import time
  4. # ことわざリスト
  5. kotowaza_list = ["さるもきからおちる", "じゃのみちはへび", "にとうおうものはいっとうもえず", "でるくいはうたれる", "はなよりだんご", "うりのつるにはなすびにならず", "いのなかのかわずたいかいにでず"]
  7. # ゲームのメインループ
  8. score = 0
  10. input("エンターキーを押してスタート！")
  12. start_time = time.time()
  14. for kotowaza in kotowaza_list:
  15.     print(f"ひらがな: {kotowaza}")
  16.     
  17.     input_text = input("ひらがなで入力してください: ")
  18.     
  19.     if input_text == kotowaza:
  20.         print("正解です！")
  21.         score += 1
  22.     else: print("不正解です。")
  24. end_time = time.time()
  26. elapsed_time = end_time - start_time
  28. if elapsed_time < 100:
  29.     print(f"ゲーム終了！正答率: {score}/7, 得点: {score}/10")
  30. else:
  31.     print("タイムアップ")
  33. # 得点に応じ再挑戦のオプションを提供
  34. retry = input("もう一度プレイしますか？ (はい/いいえ): ")
  35. if retry.lower() == "はい":
  36.     # ゲームをリセットして再挑戦
  37.     score = 0

反省点

ひらがなで入力したら正解になるはずなのに、漢字にで入力したら正解になった。 修正したかったが時間がなく妥協ため調べながら修正点を見つけていく。
