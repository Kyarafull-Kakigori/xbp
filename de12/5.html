〜簡易的なデータの分析をしよう！


今回は中学・高校の時に習ったデータの分析（平均値・中央値・最大値・最小値・標準偏差）を瞬時に出せるプログラムを作りました。

コードは下記の通りです。

  1. import numpy as np
  3. data = [9843672426472, 837492374265826348, 8847327436824, 183746825913784, 239856135476134]
  4. mean = np.mean(data)
  5. print("平均値:", mean)
  7. import numpy as np
  9. data = [9843672426472, 837492374265826348, 8847327436824, 183746825913784, 239856135476134]
  10. median = np.median(data)
  11. print("中央値:", median)
  13. import numpy as np
  15. data = [9843672426472, 837492374265826348, 8847327436824, 183746825913784, 239856135476134]
  16. std_deviation = np.std(data)
  17. print("標準偏差:", std_deviation)
  19. data = [9843672426472, 837492374265826348, 8847327436824, 183746825913784, 239856135476134]
  20. max_value = max(data)
  21. min_value = min(data)
  22. print("最大値:", max_value)
  23. print("最小値:", min_value)

・利用方法

１、アプリを起動
２、平均値・中央値・最大値・最小値・標準偏差に適当な値・適当な個数入力していく
３、実行すると求めたい値が出る
　　　

・反省点

・簡単すぎて莫大な数字でない限りこのプログラムを実行する意味がない
・簡単なものに逃げず世界各国の人口推移や気温の推移の分析など、専門的なホームページに飛ばないとわからないテーマにすればよかった
